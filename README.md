# IPL-2025-Performance-Dashboard

## 📌 Project Description

This project is an interactive **Power BI dashboard** built to analyze player and team performance in the Indian Premier League (IPL) 2025 season. It transforms raw cricket data into meaningful insights, helping users explore key metrics such as top run scorers, leading wicket-takers, and overall team performance.

## 🎯 Objectives

* Analyze batting and bowling performance
* Identify top players (Orange Cap & Purple Cap winners)
* Compare team performance across matches
* Provide interactive insights using filters and visuals

## 📊 Dashboard Features

* 🟠 **Orange Cap Winner** – Highest run scorer (Sai Sudharsan – 759 runs)
* 🟣 **Purple Cap Winner** – Highest wicket-taker (Prasidh Krishna – 25 wickets)
* 🏆 **Trophy Winner** – RCB (Royal Challengers Bangalore)
* 📈 **Runs by Batsman** – Bar chart showing top players
* 📉 **Wickets by Bowler** – Performance comparison of bowlers
* 📌 **Total Centuries** – KPI card showing overall centuries
* 🎛️ **Interactive Filters** – Teams, Venues, and Match Stages

  ## 📊 Analytical Highlights

* 🟠 **Top Batting Performance**: Sai Sudharsan emerged as the leading run scorer with 759 runs, dominating the season with consistent performances.

* 🟣 **Top Bowling Performance**: Prasidh Krishna secured the highest wickets (25), showcasing strong bowling efficiency and match impact.

* 🏆 **Team Dominance**: RCB displayed outstanding consistency across matches, leading them to win the IPL 2025 title.

* 📈 **Run Distribution Insights**: A significant gap is observed between the top 3 batsmen and the rest, indicating high dependency on key players.

* 📉 **Bowling Trends**: Wicket distribution shows a gradual decline after top bowlers, highlighting the importance of strike bowlers in matches.

* 💥 **High Impact Players**: Players contributing in both consistency and peak performance played a crucial role in team success.

* 📊 **Century Analysis**: A total of 8 centuries were recorded, reflecting strong batting performances throughout the tournament.

* 🎯 **Performance Variability**: Teams and players showed varying performance across venues and match stages, emphasizing the importance of adaptability.

* 🔍 **Key Insight**: Balanced team composition (strong batting + effective bowling) is a major factor behind winning the tournament.



## 🛠️ Tools & Technologies Used

* **Power BI** – Dashboard creation & visualization
* **DAX (Data Analysis Expressions)** – Calculations and KPIs
* **Data Modeling** – Structuring datasets for analysis


## 💡 Key Insights

* Sai Sudharsan emerged as the top run scorer of IPL 2025
* Prasidh Krishna led the bowling charts with highest wickets
* RCB performed consistently to win the championship
* The dashboard enables dynamic exploration using slicers

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use slicers to filter by team, venue, or match stage
4. Explore interactive charts and KPIs

This project showcases the power of data visualization in sports analytics by providing clear insights into IPL 2025 performance. It highlights skills in **data analysis, storytelling, and dashboard design**.
